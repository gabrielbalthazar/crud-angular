# Cadastro de Cursos - Frontend

## Descrição

Este projeto consiste em uma aplicação frontend desenvolvida em Angular para gerenciamento de cursos e aulas.

A aplicação permite cadastrar cursos e adicionar aulas vinculadas a cada curso, incluindo links de vídeos do YouTube. Também é possível realizar filtros para facilitar a busca pelos cursos cadastrados.

Este projeto consome uma API desenvolvida em Spring Boot.

## Tecnologias utilizadas

Angular
TypeScript
Angular Material
HTML
CSS

## Funcionalidades

Cadastro de cursos
Listagem de cursos
Edição de cursos
Exclusão de cursos
Cadastro de aulas vinculadas a um curso
Adição de links de vídeos do YouTube nas aulas
Filtro de cursos

## Como executar o projeto

1. Clonar o repositório

git clone https://github.com/gabrielbalthazar/crud-angular.git

2. Entrar na pasta do projeto

cd crud-angular

3. Instalar as dependências

npm install

4. Executar o projeto

npm run start
