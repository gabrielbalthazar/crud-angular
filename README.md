# Cadastro de Cursos - Frontend

## Descrição

Este projeto consiste em uma aplicação frontend desenvolvida em Angular para gerenciamento de cursos e aulas.

A aplicação permite cadastrar cursos e adicionar aulas vinculadas a cada curso, incluindo links de vídeos do YouTube.

Este projeto consome uma API desenvolvida em Spring Boot.

## Tecnologias utilizadas

Angular
TypeScript
HTML
CSS

## Funcionalidades

Cadastro de cursos
Listagem de cursos
Edição de cursos
Exclusão de cursos
Cadastro de aulas vinculadas a um curso
Adição de links de vídeos do YouTube nas aulas

## Como executar o projeto

Instale as dependências:

npm install

Execute o projeto:

npm run start
